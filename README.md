# InterWhom

## Practice. Improve. Get Hired.

InterWhom is an AI-powered interview assistant that helps candidates practice interviews, receive personalized feedback, and improve their interview skills.

## Features

- AI-powered mock interviews
- Role-based interview questions
- Answer evaluation
- Performance feedback
- Interview progress tracking

## Tech Stack

Frontend:
- React

Backend:
- FastAPI (Python)

Database:
- SQLite/PostgreSQL

AI:
- LLM API integration
